# spacesim
Ein 2D Weltraum-Simulator, welcher mithilfe von n-Body Simulation eine realistische Physik-Engine besitzt. Geschrieben mit Python und OpenGL, bietet ausgiebige Anpassungsmöglichkeiten an.

## Wie wird diese Repository genutzt?
Hier werden regelmäßig Änderungen und Updates in entsprechenden Branches (Zweige) hochgeladen. Es gibt eine master branch, worauf Changes ohne Backups erstellt werden. Für große Updates gibt es dementsprechend eigene Branches, um den Fortschritt pragmatischer festzuhalten.
